 <h2> Criando o projeto </h2>
  
  * 1 Clone do projeto
   ```sh
   git clone https://github.com/danilogomes08/teste-front-end-jr.git
   ```
   * 2 Criando projeto React
   ```sh
   npx create-react-app teste-front-end-jr
   ```

   ```sh
   cd teste-front-end-jr
   npm start
   ```
